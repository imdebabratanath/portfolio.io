#  My Projects Portfolio

Welcome to my GitHub repository!  
This repo contains three of my main projects showcasing graphics programming.

---

##  Kube (C++ / GLFW )

A modern OpenGL 3.3 project that renders a colorful rotating 3D cube.


### ✨ Features
- Manual 4x4 matrix math (model, view, projection)
- Depth testing
- Time-based glow shader
- Smooth X/Y axis rotation


##  SPlayGround (C++ / OpenGL / GLAD)

A real-time shader exploration tool built in **C++** and **OpenGL (Core Profile)**.  
Switch between **gradient**, **Mandelbrot**, **Julia**, and **plasma shaders** with interactive controls.

---

## ✨ Features
- Real-time switching between multiple shaders
- Interactive controls for shader parameters
- High-performance rendering using OpenGL core functions
- Manual shader compilation and linking
- GLAD-only setup (no GLFW/GLUT required)
